Connect your WordPress website with Wittypen's content writing marketplace using a unique API key.

== Description ==

The Wittypen Integration WordPress Plugin is designed to make it easy for you to publish content from Wittypen directly to your WordPress website. Once the plugin is installed and activated, it generates a unique API key. This API key can be entered into your Wittypen account settings to link your WordPress website.

With this plugin, you can publish content from Wittypen directly to your WordPress website.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress
2. Navigate to the 'Wittypen API Key' page in the admin menu to view your API key

== Frequently Asked Questions ==

= Where can I find my API key? =

After activating the plugin, you can find your API key on the 'Wittypen API Key' page in the sidebar.

= How do I enter my API key into Wittypen? =

The API key can be entered into your Wittypen account settings. However, if you are having trouble, please refer to the Wittypen documentation or contact Wittypen support for instructions on entering your API key.
